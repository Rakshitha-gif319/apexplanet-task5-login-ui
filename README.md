# ApexPlanet Task 5 - Login & Registration System

## 📌 Project Overview
This project is part of the **ApexPlanet Internship - Task 5**.  
It is a secure **User Login & Registration System** built using **PHP & MySQL** with a **space-themed modern UI** inspired by Apex Planet.  
It includes **password hashing**, **session management**, and a clean **dashboard interface**.

---

## 🚀 Features
- User Registration with input validation
- Secure Login with password hashing
- Glassmorphism & space-themed background design
- Session-based authentication
- Dashboard page for logged-in users
- Logout functionality

---

## 🛠️ Technologies Used
- **Frontend:** HTML, CSS (Glassmorphism UI)
- **Backend:** PHP
- **Database:** MySQL (via XAMPP / MariaDB)
- **Design:** Space-themed background from Unsplash
